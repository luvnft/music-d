# full player starter kit

The purpose of this repository is to get you up & running quickly with the Frosty SDK (frosty-sdk) for Full Music Player Apps.

## Getting Started

First, install dependencies using npm (note: `yarn add` has reported mixed results):

```bash
npm i
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.tsx`. The page auto-updates as you edit the file.
